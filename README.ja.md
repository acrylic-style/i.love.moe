# i.らぶ.moe

[English](README.md) | [日本語](README.ja.md)

**Minecraftのスクリーンショットを、ワンクリックで共有。**

i.らぶ.moeは、スクリーンショットを短い共有URLに変換するクライアント側のFabric
Modです。**F2**でスクリーンショットを撮ると、チャットにクリック可能なアップロード
ボタンが表示されます。アップロードが完了すると、短縮URLが自動的にクリップボードへ
コピーされます。

## 機能

- Minecraftからスクリーンショットを直接アップロード
- 作成された短縮URLを自動的にコピー
- 現在のサーバー名とアドレスを画像のメタデータとして記録
- ゲーム内チャットからアップロード画像を開く、コピーする、削除する
- Webサイトから画像タイトル、アルバム、タグ、お気に入り、公開範囲を管理
- URL限定公開、非公開、合言葉付き公開に対応
- Minecraftの言語設定に応じて英語と日本語を表示
- Plus利用者向けのF2自動アップロード

このModは完全にクライアント側で動作するため、サーバーへの導入は必要ありません。

## 使い方

1. Fabric Loader、Fabric APIと一緒にModを導入します。
2. **F2**でスクリーンショットを撮ります。
3. チャットメッセージの**[アップロード]**をクリックします。
4. スクリーンショットがアップロードされ、URLがクリップボードへコピーされます。

無料プランでは手動でアップロードできます。Plus利用者は自動アップロードを有効に
できます。

```text
/ilovemoe auto-upload on
```

## コマンド

```text
/ilovemoe login <email>
/ilovemoe auto-upload
/ilovemoe auto-upload on
/ilovemoe auto-upload off
```

ログインコマンドを実行すると、スクリーンショットをWeb管理画面から管理するための
マジックリンクが送信されます。

## プラン

無料プランでは手動アップロード、直近30日間に50枚までのアップロード、30日間の保存を
利用できます。

Plusには次の機能が含まれます。

- 直近30日間に500枚までアップロード
- 365日間の保存
- スクリーンショットの自動アップロード
- より多くのアルバムとアルバム内画像
- 非公開と合言葉付き公開
- タグ、お気に入り、一括整理

## プライバシーについて

スクリーンショットには、プレイヤー名、チャットメッセージ、サーバーアドレスなどの
個人情報が含まれる場合があります。アップロード前に内容を確認し、適切な公開範囲を
選んでください。
